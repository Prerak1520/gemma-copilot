# Demo Assets

These screenshots are cropped/redacted copies prepared for public posts. Use these versions instead of the original desktop screenshots.

## Recommended Order

1. `extension-analysis-result.png`
   - Alt text: Gemma Copilot Chrome extension showing a local Gemma analysis result with an ATS score, fit summary, skill gaps, feedback controls, and resume generation.
   - Caption: Gemma Copilot analyzes a job description directly from the browser and returns an ATS-aware verdict without sending career context to a cloud LLM.

2. `skill-diff.png`
   - Alt text: Gemma Copilot dashboard showing the skill prompt comparison view for the job-description analysis workflow.
   - Caption: The dashboard keeps prompt versions reviewable so feedback-driven reflection can be inspected before any skill update is promoted.

3. `resume-generator-controls.png`
   - Alt text: Gemma Copilot dashboard controls for selecting a job, choosing a resume template, and generating a tailored resume.
   - Caption: Resume generation is part of the same local workflow, turning a scored job into a tailored draft.

4. `api-local-runtime.png`
   - Alt text: Terminal output showing the Gemma Copilot API running locally with edge and deep Gemma models configured.
   - Caption: The API runs locally on port 3939 and uses Ollama-hosted Gemma models for inference.

5. `dashboard-local-runtime.png`
   - Alt text: Terminal output showing the Gemma Copilot dashboard running locally on localhost port 3940.
   - Caption: The Next.js dashboard runs locally and proxies API requests to the local Express server.
